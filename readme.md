# Dotfiles

A minimal development environment set up with some essential tools.
Only the MacOS parts that go into MacBooks are here, server and home lab
configurations are in different (private) repos. The goal is to move those
into a folder here and let MacBooks base some configurations from it but
not to overcomplicate Mac configuration given the number of applications
in use. If the number gets quite high one day, it would make sense to
move all to Nix - but the goal is not to get to that state either.

## Notes

* Theme - [Bitstream](https://github.com/s-silva/bitstream-theme), Ghostty
config has colors taken from the theme already, VSCode theme should be
installed directly from the marketplace, a development version of the
neovim theme will go in here alongside the nvim config.

* Zim - git etc. aliases were manually added to keep the consistency with
oh-my-zsh, which I used for a long time.
